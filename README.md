# FoodBuck

FoodBuck is a **static, frontend-only restaurant website** built with **Next.js, Tailwind CSS, and shadcn/ui**, featuring a premium black & orange theme.
It includes menu, shop (dummy checkout), blog, and contact pages with all fully responsive.

## Installation & Usage

```bash
git clone https://github.com/shayanbalochofficial/Food-ecommerce
cd foodbuck
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
